# zabbix-media-vkteams
Zabbix Webhook Media Type to trigger basic alerts to VK Teams (VK WorkSpace)

Template for VK Teams based on [official zabbix telegram template](https://git.zabbix.com/projects/ZBX/repos/zabbix/browse/templates/media/telegram)

## Service setup
1. Register bot: send "/newbot" to @metabot and follow instructions
2. Copy and paste the obtained token into the "Token" field above
